# **BIN()**

## **Description**

This function ``BIN()`` returns a string representation of the binary value of *N*, where *N* is a ``longlong (BIGINT)`` number. Returns ``NULL`` if *N* is *NULL*.

## **Syntax**

```
> BIN(N)
```

## **Arguments**

|  Arguments   | Description  |
|  ----  | ----  |
| N | Required. UINT Type |
