# **SHOW DATABASES**

## **Description**

Shows the list of databases that exist on the instance.

## **Syntax**
```
> SHOW DATABASES [LIKE expr | WHERE expr]
```
## **Examples**
```
> SHOW DATABASES;
+----------+
| Database |
+----------+
| default  |
| for_test |
| local    |
| ss       |
| ss1      |
| ss2      |
| ss3      |
| system   |
| test     |
+----------+
9 rows in set (0.00 sec)
```
