# **YEAR()**

## **函数说明**

`YEAR()`和`TOYEAR()`函数返回了给定日期的年份（从1000到9999）。


## **函数语法**

```
> YEAR(date)
> TOYEAR(date)
```
## **参数释义**
|  参数  | 说明  |
|  ----  | ----  |
| date  | 必要参数，需要提取年份的日期 |



## **示例**


```sql
> SELECT YEAR("2017-06-15");
2017
```

## **限制**

目前只支持`yyyy-mm-dd` 和 `yyyymmddd`的数据格式。
