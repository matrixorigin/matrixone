# **INTERVAL**

## **语法说明**

`INTERVAL` 用于日期和时间计算。

## **语法结构**

```
> INTERVAL (expr,unit)
```

## **示例**

```sql
> select INTERVAL 1 DAY + "1997-12-31";
+-------------------------------+
| interval(1, day) + 1997-12-31 |
+-------------------------------+
| 1998-01-01                    |
+-------------------------------+
1 row in set (0.01 sec)
```
