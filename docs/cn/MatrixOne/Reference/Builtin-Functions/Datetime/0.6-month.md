# **MONTH()**

## **函数说明**

返回日期的月份，对于1月到12月，返回范围为1到12；对于“0000-00-00”或“2008-00-00”等日期，返回0月份。如果 `date` 为 `NULL` 则返回 `NULL`。

## **函数语法**

```
> MONTH(date)
```

## **参数释义**

|  参数   | 说明 |
|  ----  | ----  |
| date  | 必要参数。 date 参数是合法的日期表达式。  |

## **示例**

- 示例 1:

```sql
> SELECT MONTH('2008-02-03');
+-------------------+
| month(2008-02-03) |
+-------------------+
|                 2 |
+-------------------+
1 row in set (0.02 sec)
```
