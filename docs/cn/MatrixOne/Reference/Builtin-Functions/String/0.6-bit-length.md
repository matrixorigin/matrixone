# **BIT_LENGTH()**

## **函数说明**

返回字符串 str 的长度，单位为 bit。如果 str 为 `NULL` 则返回 `NULL`。

## **语法说明**

```
> BIT_LENGTH(str)
```

## **参数释义**

|  参数   | 说明  |
|  ----  | ----  |
| str | 必要参数，想要计算长度的字符串 |

## **示例**

```SQL
> SELECT BIT_LENGTH('text');
+------------------+
| bit_length(text) |
+------------------+
|               32 |
+------------------+
1 row in set (0.00 sec)
```
